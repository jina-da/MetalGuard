[ 구동 환경 ]

-. 32bit OS

-. Pylon SDK 3.x  이상 

-. Visual Studio 2008 이상

-. Camera관련 코드는 CameraManager class 에 있습니다.

 [ 프로그램 구동 순서 ]

**Heartbeat time 설정은 아래 두가지 방법 입니다.

[ 방법 1]-> Registry 설정

#if _DEBUG

     _putenv("PYLON_GIGE_HEARTBEAT=3000");  /* set to 3000 ms = 3sec */

#else if 

     _putenv("PYLON_GIGE_HEARTBEAT=3000");  /* set to 3000 ms = 3sec */

#endif

 

[ 방법 2] -> 코드 설정 CameraManager.cpp -> OpenCamera 함수

 

//Debug 모드중BreakPoint 걸어서코드확인시HeartbeatTime시간동안코드이동이없는경우카메라연결이끊어짐

//release 배포시에는시간을3000ms(3초)로설정

m_pHeartbeatTimeout[ nCamIndex ] = m_pCamera[ nCamIndex ].GetTLNodeMap().GetNode("HeartbeatTimeout");

m_pHeartbeatTimeout[ nCamIndex ]->SetValue(3000);

 

 

1. CameraInfo.txt에서 카메라 serial number 수정 [ 보유하고 있는 제품 serial number로 수정 카메라 한대인 경우 CAM1 만 수정 ]

2. Find camera

3. 리스트에서 카메라 선택                   

4. Open Camera

5. Connect Camera -> AOI 및 이미지 포맷 설정  [ 기본 640 x 480 , Mono8 or Bayer]

   -> Trigger 설정 [ Software trigger모드 ]

6. Live Grab  [ Grab 완료후에 CameraManager class OnImageGrabbed call back함수에서 이미지 conversion 함 ]

   -> S/w Trigger 모드인 경우 Software Trigger Execute 로 영상 취득

7. SaveImage
